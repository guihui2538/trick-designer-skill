# Trick Designer

> 推理小说诡计设计工具 | 兼容 Claude Code, OpenClaw, Codex, Gemini CLI

## 简介

专门用于设计推理小说诡计的工具，帮助创作者从不可能的谜面出发，通过反直觉的心理学盲区或物理科学原理设计作弊点，并形成逻辑闭环。

---

## 核心功能

- **双智能体模式**：设计者+侦探，迭代优化诡计
- **难度分级**：入门/进阶/大师
- **自动联网研究**：输入设定后自动搜索相关作品
- **6大谜面类型**：密室/不在场证明/身份替换/设定系变格/软科幻

---

## 支持的诡计类型

| 类型 | 说明 |
|------|------|
| 空间不可能 | 密室、不可能进入 |
| 时间不可能 | 完美不在场证明、延时犯罪 |
| 身份不可能 | 双胞胎、假死、身份替换 |
| 逻辑不可能 | 矛盾事实同时为真 |
| 设定系变格 | 超自然伪装、伪轮回、伪魔法 |
| 软科幻 | 记忆移植、时间悖论、虚拟世界 |

---

## 设计流程

```
1. 联网搜索相关作品
2. 保存作品到经典作品库
3. 选择谜面类型（6大类）
4. 选择难度（入门/进阶/大师）
5. 选择作弊点方向（心理学/物理科学）
6. 构建模板
7. 设计反转
8. 侦探测试（双智能体）
9. 检验公平性
10. 评分系统
11. 决策（pass/优化/使用）
12. 保存诡计（注明参考来源）
```

### 评分标准

| 维度 | 分数 | 说明 |
|------|------|------|
| 创新性 | 1-10 | 是否跳出常见模式 |
| 逻辑性 | 1-10 | 是否自圆其说 |
| 公平性 | 1-10 | 线索是否藏好 |
| 可行性 | 1-10 | 原理是否成立 |
| 反转度 | 1-10 | 是否有意外惊喜 |

### 决策

- 总分<30分：pass，重新生成
- 总分30-60分：优化后使用
- 总分>60分：优秀，使用

## 随机创新模式

选择2-4个元素随机组合：

- 设定元素：限定条件、特殊职业、时间/空间限制
- 科技元素：AI监控、AR/VR、生物技术、区块链
- 社会元素：网络暴力、直播犯罪、社交网络
- 文化元素：传统习俗、宗教信仰、游戏文化
- 反直觉元素：视觉盲区、时间感知偏差

## 使用示例

**用户**：我想设计一个密室杀人的诡计

**AI流程**：
1. 联网搜索密室相关经典作品
2. 保存好的新作品到作品库
3. 选择类型：空间不可能 → 密室
4. 选择难度：进阶
5. 选择方向：物理 → 光学
6. 选择模板：物理密室 → 镜子密室
7. 设计反转：密室实际没锁
8. 侦探测试：尝试破解，找漏洞
9. 检验公平性
10. 评分并迭代
11. 保存诡计 → 注明参考来源
