## 📖 简介

专门用于设计推理小说诡计的工具，帮助创作者从不可能的谜面出发，通过反直觉的心理学盲区或物理科学原理设计作弊点，并形成逻辑闭环。

**一句话概括**：让AI扮演"诡计设计者"和"侦探"两个角色，通过双智能体迭代优化，设计出创新的推理小说诡计。

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🔄 **双智能体模式** | 设计者负责构建，侦探负责破解，通过对抗迭代优化 |
| 📊 **难度分级** | 入门/进阶/大师三档，适应不同水平创作者 |
| 🌐 **自动联网研究** | 实时搜索最新作品和诡计趋势 |
| 📚 **作品库管理** | 自动保存经典作品和设计的诡计，支持参考 |
| 🎲 **随机创新** | 随机组合科技/社会/文化元素，创造新诡计 |
| ✅ **评分系统** | 创新性/逻辑性/公平性/可行性/反转度五维评分 |

## 🔥 设计流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                        🎭 诡计设计完整流程                            │
└─────────────────────────────────────────────────────────────────────┘

  ① 联网搜索
     │
     ▼
  ② 保存作品 → 经典作品库
     │
     ▼
  ③ 选择谜面类型（6大类）
     │
     ├── 空间不可能 → 密室/不可能进入
     ├── 时间不可能 → 不在场证明/延时犯罪
     ├── 身份不可能 → 双胞胎/假死/身份替换
     ├── 逻辑不可能 → 矛盾事实
     ├── 设定系变格 → 超自然伪装
     └── 软科幻 → 记忆移植/时间悖论
     │
     ▼
  ④ 选择难度 → 入门/进阶/大师
     │
     ▼
  ⑤ 选择作弊点方向
     │
     ├── 心理学：认知盲区/社会工程/感知操控
     └── 物理科学：力学/光学/化学/生物
     │
     ▼
  ⑥ 构建模板 → 密室模板/不在场证明模板/身份模板
     │
     ▼
  ⑦ 反转设计 → 设定反转/身份反转/手法反转/动机反转
     │
     ▼
  ⑧ 侦探测试 ← 双智能体
     │
     ├── 侦探尝试破解
     ├── 找出明显线索
     ├── 检验反转惊喜感
     └── 记录漏洞
     │
     ▼
  ⑨ 检验公平性 → 线索隐藏/反转感/动机合理/避开定式
     │
     ▼
  ⑩ 评分系统（5维度）
     │
     ├── 创新性（1-10）
     ├── 逻辑性（1-10）
     ├── 公平性（1-10）
     ├── 可行性（1-10）
     └── 反转度（1-10）
     │
     ▼
  ⑪ 决策
     │
     ├── <30分 → ❌ pass，重新生成
     ├── 30-60分 → 📝 优化后使用
     └── >60分 → ✅ 优秀，使用
     │
     ▼
  ⑫ 保存诡计 → 注明参考来源
```

## 🎲 随机创新模式

当你想不出新idea时，可以启用随机创新：

```bash
# 随机选择2-4个元素组合
设定元素 + 科技元素 + 社会元素 + 文化元素 + 反直觉元素
```

**示例组合**：
- 限定条件（只能在直播中犯罪）+ AI监控 + 时间感知偏差
- 结果：密室+直播+时间差不在场证明

## 📝 使用示例

**用户**："我想设计一个密室杀人的诡计"

**AI流程**：
1. 🔍 联网搜索密室相关经典作品
2. 💾 保存好的新作品到作品库
3. 🏠 选择类型：空间不可能 → 密室
4. 📊 选择难度：进阶
5. 🔬 选择方向：物理 → 光学
6. 📋 选择模板：物理密室 → 镜子密室
7. 🔄 设计反转：密室实际没锁
8. 🔍 侦探测试：尝试破解，找漏洞
9. ✅ 检验公平性
10. 📊 评分并迭代
11. 💾 保存诡计 → **注明参考来源**（如参考了《钟表馆》的密室重建）
