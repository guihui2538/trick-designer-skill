## 简介

专门用于设计推理小说诡计的工具，帮助创作者从不可能的谜面出发，通过反直觉的心理学盲区或物理科学原理设计作弊点，并形成逻辑闭环。

## 核心功能

- **双智能体模式**：设计者+侦探，迭代优化诡计
- **难度分级**：入门/进阶/大师
- **自动联网研究**：输入设定后自动搜索相关作品
- **6大谜面类型**：密室/不在场证明/身份替换/设定系变格/软科幻

## 支持的诡计类型

| 类型 | 说明 |
|------|------|
| 空间不可能 | 密室、不可能进入 |
| 时间不可能 | 完美不在场证明、延时犯罪 |
| 身份不可能 | 双胞胎、假死、身份替换 |
| 逻辑不可能 | 矛盾事实同时为真 |
| 设定系变格 | 超自然伪装、伪轮回、伪魔法 |
| 软科幻 | 记忆移植、时间悖论、虚拟世界 |

## 使用方式

1. 选择谜面类型
2. 选择难度（入门/进阶/大师）
3. 选择作弊点方向（心理学/物理科学）
4. 选择具体模板
5. 设计反转
6. 双智能体测试（设计者设计→侦探破解→优化）
7. 检验公平性
