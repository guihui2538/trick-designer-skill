> 兼容平台：Claude Code, OpenClaw, Codex, Gemini CLI

## 随机元素类型

### 1. 设定元素（随机组合）

| 元素 | 说明 | 组合效果 |
|------|------|---------|
| 限定条件 | 只能在这个场合犯罪 | 密室+限定 |
| 特殊职业 | 凶手有特殊身份 | 医生/警察/主播 |
| 时间限制 | 只有这段时间 | 节日/活动/日食 |
| 空间限制 | 只能在这个地点 | 交通工具/电梯 |
| 工具限制 | 只能用这个工具 | 某种毒药/某种装置 |
| 规则限制 | 必须按规则进行 | 游戏/比赛/仪式 |

## 随机组合模板

### 模板1：限定+科技

```
随机元素1：限定条件 → "只能在高铁上犯罪"
随机元素2：科技元素 → "全程直播"
组合效果：密室+直播+限定
```

### 模板2：社会+文化

组合效果：他杀伪装成意外+仪式

## 随机生成器使用

### 步骤1：选择组合数量

- 2个元素：基础创新
- 3个元素：进阶创新
- 4个元素：大师级创新

### 步骤2：随机选择元素

从上述5类元素中各选1个：
- 1个设定元素
- 1个科技元素
- 1个社会元素
- 1个文化元素
- 1个反直觉元素

### 步骤3：分析组合效果

根据组合思考：
- 什么"不可能"能被创造？
- 有什么新诡计可以设计？
- 需要什么支撑条件？

### 步骤4：双智能体评判

设计者完成初稿后，侦探判断：
- 创新性：是否够"新"？
- 逻辑性：是否自圆其说？
- 可行性：元素是否合理？
- 反转度：是否有意外惊喜？

评分标准：
- <30分：pass，重新生成
- 30-60分：优化后使用
- >60分：优秀，使用
