## 密室类

### 物理密室

#### 《占星术杀人事件》- 岛田庄司
**核心诡计**：六人分饰多角

**原理拆解**：
1. 表面：六名少女在不同地点同时被杀 → 不可能
2. 作弊点：六人中只有一人真正死亡，其他五人被绑架后被"扮演"
3. 支撑：凶手是其中一人的母亲，有动机和机会
4. 线索：少女们的证词有微妙矛盾
5. 反转：读者以为六人都是受害者 → 实际只有一人

#### 《钟表馆事件》- 绫辻行人
**核心诡计**：密室重建

**原理拆解**：
1. 表面：密室门从内锁，现场无出口
2. 作弊点：凶手先制造密室，再从外部重建密室（调包）
3. 支撑：钟表馆内部多重构造，有暗道
4. 线索：钟表的时间暗示

#### 《斜屋犯罪》- 岛田庄司
**核心诡计**：密室+不可能消失

**原理拆解**：
1. 密室中被害人消失
2. 利用建筑倾斜造成的视觉盲区
3. 密室本身就是障眼法

### 心理密室

#### 《第四十二号避难所》- 岛田庄司
**核心诡计**：心理密室（门没锁）

**原理拆解**：
1. 表面：密室不可能进入
2. 作弊点：门根本没锁，是心理暗示让所有人认为门锁着
3. 线索：没人真正检查过门锁

## 游戏/动画诡计库

### 视觉小说/游戏

#### 《Ever17 -时空胶囊-》
**核心诡计**：双重时间线叙事

**原理拆解**：
1. 表面：玩家视角是"少年"
2. 作弊点：实际视角是"少女"，玩家被性别误导
3. 支撑：角色立绘和名字的性别暗示
4. 线索：语音和部分文本的微妙差异
5. 反转：通关后重新理解整个故事

**借鉴要点**：

- 视角转换型叙诡：利用玩家代入感制造盲区
- 多周目真相：二周目才能理解一周目的"不合理"
- 角色身份的双重性

#### 《428被封锁的涩谷》
**核心诡计**：多视角交织+选择分支

**原理拆解**：
1. 表面：5个角色各自独立的故事
2. 作弊点：5条线实际是同一事件的不同时间点
3. 支撑：某个角色的选择会影响其他角色的命运
4. 线索：5条线中的"巧合"细节

#### 《逆转裁判》
**核心诡计**：证言矛盾+证据推翻

**原理拆解**：
1. 表面：证人证词看似可信
2. 作弊点：证词中有逻辑矛盾
3. 支撑：利用"异物"（异物指与现场不符的物品）推翻证词
4. 线索：证人的微表情和小动作

#### 《雷顿教授与最后的时光》
**核心诡计**：时间循环+记忆重置

**原理拆解**：
1. 表面：每一天都是新的一天
2. 作弊点：主角保留记忆，但其他人被重置
3. 线索：细节的不连贯

### 游戏/动画诡计设计技巧

#### 1. 利用玩家/观众视角

```
玩家视角 = 主角视角
→ 但主角不是"表面主角"
→ 利用身份转换制造反转
```

#### 2. 利用多结局/多周目

```
一周目：看似正常的结局
二周目：发现一周目的"不合理"
三周目：揭示真正的真相
```

#### 3. 利用交互性

```
选项A → 导向错误结论
选项B → 导向另一条错误结论
隐藏选项 → 才是正确答案
```

#### 4. 利用沉浸感

```
玩家会注意"自己的选择"
→ 忽略"剧情的必然性"
→ 利用必然事件制造反转
```

## 设定系变格类（超自然悬疑）

### 核心特征

| 特征 | 说明 |
|------|------|
| **伪超自然** | 表面是超自然现象，实际是人为 |
| **多重真相** | 表面解释+隐藏的真实 |
| **设定依托** | 依托神秘学/民俗/科学设定 |
| **氛围营造** | 利用恐怖氛围制造心理盲区 |

### 《寒蝉鸣泣之时》系列

**核心诡计**：轮回+解谜

**原理拆解**：
1. 表面："鬼隐"现象，，村民相继死亡/失踪
2. 作弊点：实际上是人为的"剧本"，有人故意制造恐怖
3. 支撑：利用村民的迷信和恐惧心理
4. 线索：，每年重复的模式，唯一的"局外人"
5. 反转：表面是超自然，实际是"为了拯救村民的谋杀"

**借鉴要点**：

- **轮回外壳**：时间循环是假，剧本才是真
- **自相矛盾**：超自然解释无法自圆其说 → 人为
- **氛围欺骗**：利用恐怖氛围让读者忽略逻辑漏洞

---

### 《海猫鸣泣之时》系列

**核心诡计**：密室+遗产争夺+魔法设定

**原理拆解**：
1. 表面：六轩岛上的"魔女"诅咒，魔术手法不可能
2. 作弊点："魔术"实际上是利用现场条件的"犯罪"
3. 支撑：复杂的遗产争夺背景，多个嫌疑人
4. 线索：魔术手法的"不可能"都有科学/人为解释
5. 反转：EP4-EP6的多重真相

**借鉴要点**：

- **复杂设定**：引入"魔法"体系但内部自洽
- **多重解答**：同一个案件有多种"正确"解答
- **证人筛选**：利用角色立场筛选"可信"证词

---

### 《混沌之子》系列

**核心诡计**：都市传说+连续自杀

**借鉴要点**：

- **现代背景**：利用社交媒体/网络
- **暗示操控**：集体无意识的操控
- **多重真相**：表面超自然 → 实际心理暗示 → 实际有人为

### 设定系变格设计技巧

#### 1. 设计"伪超自然"设定

```
步骤1：设计一个看似超自然的"规则"
步骤2：让这个规则有"漏洞"（不可能完美）
步骤3：让人利用这些漏洞制造"超自然现象"
步骤4：让"破解"实际上是人為的
```

#### 2. 制造多重真相

```
表面真相：这是诅咒/魔法/轮回
第二层真相：这是心理暗示/社会事件
第三层真相：这是人为的犯罪
```

#### 3. 利用氛围盲区

```
读者在恐怖氛围中
→ 更容易接受"超自然解释"
→ 忽略逻辑漏洞
→ 利用这一点制造反转
```

#### 4. 设定与诡计结合

```
设定1：诅咒→实际是定时装置
设定2：轮回→实际是剧本重演
设定3：魔法→实际是魔术手法
```

### 《流景之海》- Aporia（阿普利亚）

**核心诡计**：记忆移植+身份认知

**原理拆解**：
1. 表面：主角拥有"另一个人的记忆"
2. 作弊点：记忆移植技术+认知操控让自己"以为"是另一个人
3. 支撑：记忆可以被篡改/植入
4. 线索：记忆的"断层"和"矛盾"
5. 反转：以为自己是A，实际是B

**借鉴要点**：

- **记忆不可靠**：我们的"记忆"可能是被植入的
- **身份认知**：自我认知可以被操控
- **技术边界**：探讨记忆移植的伦理和边界

### 软科幻设计技巧

#### 1. 设定先行

```
步骤1：设计一个科幻"设定"（记忆移植/时间机器/虚拟现实）
步骤2：在这个设定下，可能发生什么"不可能"？
步骤3：设计诡计利用这个"不可能"
步骤4：确保设定内部自洽
```

#### 2. 技术与人性结合

```
技术设定 → 产生"不可能"
↓ 
人性和诡计 → 解决"不可能"
↓
留下"科技伦理"思考
```

#### 3. 多重解释

```
表面：这是记忆移植
↓
第一层：记忆是假的
↓
第二层：记忆是真的，但"本人"不是"自己"
↓
第三层：自我认知本身就是"被设计"的
```

#### 4. 利用技术漏洞

```
设定1：记忆可被读取 → 被利用制造"证据"
设定2：时间可被穿越 → 产生悖论
设定3：虚拟可影响现实 → 制造"不可能犯罪"
```

---

### 软科幻设计检查清单

- [ ] 科幻设定是否内部自洽？
- [ ] "不可能"是否有技术解释？
- [ ] 人性部分是否足够精彩？
- [ ] 是否有反转空间？
- [ ] 科技是否服务剧情而非炫技？
