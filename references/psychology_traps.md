# 心理学盲区库（含实际应用案例）

> 兼容平台：Claude Code, OpenClaw, Codex

## 认知盲区

### 注意力偏向

**鸡尾酒会效应**

- 现象：在嘈杂环境中只关注自己的名字
- 应用案例：在集体聚会中，凶手假装不经意叫出被害人的名字，被害人转身时凶手已完成动作
- 推理小说案例：《ABC谋杀案》利用注意力分散

**变化盲视**

- 现象：对缓慢变化视而不见
- 应用案例：凶手慢慢改变现场某物（如挂钟时间），侦探不会注意到
- 设计提示：设计渐进式变化，让"不可能"在眼皮底下发生

**返回抑制**

- 现象：忽略曾注意过的位置
- 应用案例：凶手把证据藏在侦探刚刚检查过的地方

### 记忆偏差

**事后聪明偏差**

- 现象：事后认为结果"显而易见"
- 应用案例：凶手故意留下"明显"线索，事后让人觉得自己"应该想到"
- 设计提示：设计看似"马后炮"的线索

**记忆歪曲**

- 现象：根据后续信息改写记忆
- 应用案例：凶手先提供错误信息，再给出"正确"信息，被害人的记忆会被改写
- 推理小说案例：《罗杰疑案》

**舌尖现象**

- 现象：知道但说不出
- 应用案例：证人说"我知道是谁，但说不出"

### 确认偏误

**选择性搜索**

- 现象：只找支持自己观点的证据
- 应用案例：侦探找到支持自己理论的证据，忽略矛盾证据
- 设计提示：给侦探留一条"完美"的推理路径，但路上有盲点

**光环效应**

- 现象：因优点掩盖缺点
- 应用案例：外表善良的人不会被怀疑
- 设计提示：利用正面形象制造不在场证明

## 社会工程

### 权威效应

**制服欺骗**

- 现象：穿制服就能获得信任
- 应用案例：凶手假扮医生、警员、快递员进入现场
- 案例：《占星术》中母亲假扮各种角色

**头衔降低警惕**

- 现象：有头衔的人说话更可信
- 应用案例：凶手有"专家"身份，其证词更可信

### 群体压力

**多数人意见**

- 现象：多数人意见会压倒少数
- 应用案例：集体作伪证，每个人都跟着说"是"

**从众心理**

- 现象："大家都在做"的从众心理
- 应用案例：暴风雪山庄中，凶手利用恐惧让众人互相猜忌

### 信任漏洞

**熟人信任**

- 现象：熟人更容易被信任
- 应用案例：凶手是被害人的老朋友/亲戚/医生

**紧急情况降低警惕**

- 现象：紧急情况会降低警惕
- 应用案例：利用火灾、急救等紧急情况进入现场

## 感知操控

### 视觉错位

**空间关系错误**

- 现象：对空间关系理解错误
- 应用案例：利用镜子、玻璃制造"看起来在那边"的假象
- 案例：《钟表馆》中的影子欺骗

**遮挡关系误导**

- 现象：遮挡关系误导
- 应用案例：利用窗帘、屏风遮挡关键动作

**比例大小误判**

- 现象：比例和大小误判
- 应用案例：利用玩具、模型制造"不可能"的现场

### 声音误导

**回声定位错误**

- 现象：回声定位错误
- 应用案例：声音实际从别处传来

**声音方向判断失误**

- 现象：声音方向判断失误
- 应用案例：利用录音、设备制造声音

### 时间感知

**紧张时时间变快**

- 现象：紧张时时间变快
- 应用案例：利用紧张情绪制造时间错觉

**专注时忽略时间流逝**

- 现象：专注时忽略时间流逝
- 应用案例：让侦探专注于某事，忽略真正的时间

## 心理学诡计设计模板

### 模板1：注意力转移

```
步骤1：设计一个"吸引注意力"的事件
步骤2：在此事件发生同时，完成真正的犯罪
步骤3：利用返回抑制，忽略关键细节
```

### 模板2：记忆篡改

```
步骤1：在犯罪前提供错误信息
步骤2：犯罪发生后提供"正确"信息
步骤3：利用记忆歪曲让证人"记住"假象
```

### 模板3：权威伪装

```
步骤1：设计一个权威身份（医生/警察/专家）
步骤2：利用权威身份获得信任和便利
步骤3：利用权威身份制造伪证
```

### 模板4：群体误导

```
步骤1：制造紧迫感/恐惧感
步骤2：让众人互相猜忌
步骤3：利用群体压力让关键证人"随大流"
```
