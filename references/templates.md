# 诡计设计模板

> 兼容平台：Claude Code, OpenClaw, Codex

## 密室设计模板

### 模板A：物理密室（机械）

```
【类型】门锁密室
【谜面】门从内锁，被害人死于密室内
【作弊点】门锁被人为制造"假锁"状态
【支撑条件】
  - 凶手提前进入密室
  - 使用特殊装置制造"已锁"假象
  - 密码锁/机械锁可从外调整
【线索设计】
  - 锁具的微小异常
  - 门缝的痕迹
【反转设计】密室根本没锁，是心理暗示
```

```
【类型】窗户密室
【谜面】窗户锁着，被害人死于室内
【作弊点】窗户锁可以被远程操作
【支撑条件】
  - 窗户有外部开关/遥控锁
  - 利用园艺工具/长杆操作
【线索设计】
  - 窗外的痕迹
  - 工具留下的微痕
```

### 模板B：心理密室

```
【类型】门没锁
【谜面】所有人都以为门锁着，实际上没锁
【作弊点】门锁损坏/被打开但看起来像锁着
【支撑条件】
  - 利用心理暗示（门锁坏了但没人修）
  - 光线昏暗看不清
【线索设计】
  - 门锁的实际状态
  - 其他人的证词矛盾
```

### 模板C：时刻密室

```
【类型】瞬间密室
【谜面】某一时刻过后，密室形成
【作弊点】利用时刻制造密室
【支撑条件】
  - 定时装置
  - 时间差设计
【线索设计】
  - 时钟/计时器的异常
```

## 不在场证明设计模板

### 模板A：时间差不在场

```
【类型】伪造死亡时间
【谜面】嫌疑人不在现场，但实际死亡时间不同
【作弊点】利用死亡时间判定错误
【支撑条件】
  - 尸体状态判断错误
  - 温度/腐败程度误导
【线索设计】
  - 尸斑/体温的微妙异常
  - 胃内容物时间推断
【反转设计】死亡时间确实，但凶手利用时间差
```

### 模板B：空间误导

```
【类型】远程制造证据
【谜面】证据在案发现场，但凶手不在
【作弊点】利用他人/工具制造证据
【支撑条件】
  - 委托第三方
  - 利用快递/物流
【线索设计】
  - 物流记录
  - 第三方证词
```

### 模板C：心理不在场

```
【类型】记忆偏差
【谜面】证人的记忆与实际不符
【作弊点】利用心理盲区制造虚假证词
【支撑条件】
  - 事前误导
  - 事后强化
  - 环境暗示
【线索设计】
  - 证词的细节矛盾
  - 多个证词的差异
```

## 身份替换设计模板

### 模板A：双胞胎/多胞胎

```
【类型】多胞胎身份混淆
【谜面】两人是双胞胎，身份混淆
【作弊点】利用双胞胎身份制造"不可能"
【支撑条件】
  - 双胞胎外貌相似
  - 故意制造身份混淆
【线索设计】
  - 身体细节差异（胎记/疤痕）
  - 行为习惯差异
【反转设计】不是双胞胎，而是三胞胎/四胞胎
```

### 模板B：变性/化妆

```
【类型】性别转换
【谜面】男女身份混淆
【作弊点】通过化妆/变性改变外观
【支撑条件】
  - 化妆技术/医学手段
  - 长期准备
【线索设计】
  - 声音/体态的微妙差异
  - 医学记录
```

### 模板C：认知错误

```
【类型】角色认知错误
【谜面】读者/侦探认知中的角色与实际不符
【作弊点】利用叙述性诡计
【支撑条件】
  - 叙述者身份
  - 误导性描述
【线索设计】
  - 叙述中的细节矛盾
  - 性别/年龄暗示
```

## 组合模板

### 密室+不在场证明

```
步骤1：设计密室（空间不可能）
步骤2：同时设计不在场证明（时间不可能）
步骤3：两者都需要反转
步骤4：线索需要同时满足两个条件
```

### 身份+不在场证明

```
步骤1：设计身份替换（身份不可能）
步骤2：设计完美不在场证明（时间不可能）
步骤3：身份替换帮助不在场证明
步骤4：线索同时指向两个反转
```

### 叙诡+密室+反转

```
步骤1：利用叙述性诡计制造身份盲区
步骤2：设计物理密室增加难度
步骤3：设计多层反转
步骤4：线索分散在不同章节
```
