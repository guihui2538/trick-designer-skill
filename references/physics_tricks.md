# 物理科学诡计库（含实际应用案例）

> 兼容平台：Claude Code, OpenClaw, Codex

## 力学

### 重力

**坠落时间计算错误**

- 现象：从高处坠落的时间比预估短
- 应用案例：利用高楼/悬崖制造"不可能时间"的密室
- 案例：《斜屋犯罪》中的坠落设计

**重力辅助移动物体**

- 现象：利用重力可以无声移动重物
- 应用案例：用滑轮/轨道系统移动家具制造密室

**重心变化造成假象**

- 现象：重心转移造成"不可能"的视觉效果
- 应用案例：利用重心制造"悬空""漂浮"的假象

### 惯性

**物体继续运动造成时间差**

- 现象：物体移除后运动继续
- 应用案例：凶手离开后，物体（如窗帘）继续摆动

**突然加速/减速的错觉**

- 现象：运动状态的突然变化造成错觉
- 应用案例：利用运动错觉制造"消失"

### 浮力

**水中密室**

- 现象：水下空间可以隐藏物体/人
- 应用案例：利用泳池、浴缸制造"水下密室"

**漂浮物的时间差**

- 现象：漂浮物可以制造"时间差"
- 应用案例：利用漂浮物延迟发现尸体

## 光学

### 反射

**镜子造成的假位置**

- 现象：镜中像与实际位置不同
- 应用案例：利用镜子让凶手"出现在"不可能的位置
- 设计要点：镜子角度、视线盲区

**水面反射**

- 现象：水面反射制造"倒影"
- 应用案例：利用池塘、水潭制造假象

### 折射

**水中物体位置错位**

- 现象：水中物体位置看起来比实际高
- 应用案例：利用水制造"不可能看到"的位置

**玻璃折射**

- 现象：玻璃折射改变视觉位置
- 应用案例：利用厚玻璃/弧形玻璃制造视觉错误

### 影子欺骗

**影子与实物分离**

- 现象：影子可以独立于实物
- 应用案例：利用影子制造"无影人"

**影子方向误导**

- 现象：光源变化导致影子方向改变
- 应用案例：利用多光源制造时间错觉

**影子消失**

- 现象：特定角度光线可使影子消失
- 应用案例：利用影子消失制造"不在场证明"

### 其他光学

**紫外线/红外线不可见**

- 现象：某些光人眼不可见
- 应用案例：利用紫外线墨水书写隐形信息

**偏振光**

- 现象：偏振光滤光特性
- 应用案例：利用偏振光制造"隐藏图像"

## 化学

### 延迟反应

**定时释放毒气**

- 现象：化学反应可以延迟释放
- 设计：定时装置+毒气
- 案例：《圣女的救济》中的毒杀

**化学反应发热/发冷**

- 现象：化学反应伴随温度变化
- 应用案例：利用化学反应制造"冰块陷阱"

**慢性中毒**

- 现象：毒素在体内积累
- 应用案例：长期微量下毒

### 气体扩散

**气味在密闭空间扩散**

- 现象：气体扩散可被利用
- 应用案例：利用气体扩散方向判断时间

**挥发性液体**

- 现象：某些液体易挥发
- 应用案例：利用挥发清除气味

### 腐蚀

**缓慢腐蚀**

- 现象：某些物质可缓慢腐蚀
- 应用案例：利用缓慢腐蚀消除证据

## 生物

### 指纹

**戴手套无法获取**

- 现象：戴手套不留下指纹
- 应用案例：凶手戴手套处理现场

**指纹被破坏**

- 现象：指纹可以被破坏
- 应用案例：凶手故意破坏自己指纹

**指纹转移**

- 现象：指纹可以从一处转移到另一处
- 应用案例：利用手套转移指纹陷害他人

- 应用案例：利用时间/温度降解DNA

**混合DNA混淆**

- 现象：多人DNA混合
- 应用案例：利用群体环境混淆DNA

### 血迹

**血迹形状误导**

- 现象：血迹形状可被制造
- 应用案例：利用血迹形状制造"伪造现场"

**动物血 vs 人血**

- 现象：动物血可替代人血
- 应用案例：利用动物血制造不在场证明

## 物理诡计设计模板

### 模板1：光学密室

```
步骤1：选择光源（自然光/人工光）
步骤2：设计镜子/玻璃/水面的反射/折射
步骤3：利用视觉盲区制造"看不到"
步骤4：让影子与实物分离
```

### 模板2：时间毒杀

```
步骤1：选择可延迟发作的毒物
步骤2：设计"看似正常"的投毒方式
步骤3：利用化学反应延迟释放
步骤4：制造"不可能"的不在场证明
```

### 模板3：密室重建

```
步骤1：设计初始密室状态
步骤2：制造"二次密室"（调包）
步骤3：利用建筑结构/光学错觉
步骤4：隐藏真正的进出通道
```

### 模板4：指纹陷阱

```
步骤1：获取目标指纹（公开物品）
步骤2：转移到犯罪现场
步骤3：制造指纹"应该存在"的假象
步骤4：利用指纹破坏/混淆
```
