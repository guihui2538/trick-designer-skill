# 思维定式避坑指南

> 兼容平台：Claude Code, OpenClaw, Codex

## 常见思维定式

### 密室思维定式

| 定式 | 问题 | 反转思路 |
|------|------|---------|
| 默认门从内锁 | 忽视门可能没锁 | 心理密室 |
| 假设窗户锁着 | 忽视窗户可从外开 | 利用外部工具 |
| 忽略通风管道 | 忽视管道可进出 | 利用通风管道 |
| 忘记门缝 | 忽视门缝可插入工具 | 利用细长工具 |

### 不在场证明定式

| 定式 | 问题 | 反转思路 |
|------|------|---------|
| 假设证人可靠 | 忽视证人可能看错/说谎 | 记忆偏差 |
| 忽略时间误差 | 忽视时钟可做手脚 | 时钟做假 |
| 忘记远程证据 | 忽视证据可远程制造 | 委托第三方 |

### 身份定式

| 定式 | 问题 | 反转思路 |
|------|------|---------|
| 假设人脸识别可靠 | 忽视化妆/整形 | 化妆/变性 |
| 忽略双胞胎 | 忽视多胞胎 | 多胞胎 |
| 忘记声音可伪装 | 忽视声音相似 | 声音伪装 |

### 动机定式

| 定式 | 问题 | 反转思路 |
|------|------|---------|
| 只关注仇杀 | 忽视其他动机 | 无差别杀人 |
| 忽略意外 | 忽视意外可能 | 制造意外 |
| 假设有动机 | 忽视无动机 | 随机杀人 |

## 避开定式的方法

### 逆向思考

每次设计后自问：

- 凶手如果是目击者会怎样？
- 如果密室不是"密室"会怎样？
- 如果不在场证明是伪造的会怎样？
- 如果凶手是"不可能的人"会怎样？

### 增加变量

| 方法 | 效果 |
|------|------|
| 加一个看似无关的人 | 增加嫌疑人范围 |
| 加一个不可能的场所 | 制造新的谜题 |
| 加一个不可能的时间 | 增加时间复杂度 |
| 加一个假受害者 | 增加反转空间 |

### 组合变形

| 组合 | 效果 |
|------|------|
| 物理密室+心理密室 | 增加破解难度 |
| 不在场证明+身份替换 | 双重误导 |
| 密室+毒杀+不在场证明 | 三重诡计 |

## 定式检查清单

设计完成后检查：

- [ ] 是否有"显而易见"的答案？→ 尝试反转
- [ ] 密室的门锁是否被检查过？
- [ ] 证人的证词是否可靠？
- [ ] 时钟时间是否被验证？
- [ ] 是否有双胞胎/多胞胎可能？
- [ ] 动机是否过于"明显"？
- [ ] 是否有"不可能的人"被排除？
- [ ] 是否有物理盲点被忽视？

## 经典定式反转案例

### 《名侦探的守则》- 东野圭吾

- 定式：密室=门锁着
- 反转：密室根本没锁

### 《占星术杀人事件》- 岛田庄司

- 定式：六人=六具尸体
- 反转：实际只有一人死亡

### 《嫌疑人X的献身》- 东野圭吾

- 定式：流浪汉=无关人物
- 反转：流浪汉是关键证据

### 《罗杰疑案》- 阿加莎

- 定式：叙述者=旁观者
- 反转：叙述者=凶手
