# 反转设计指南

> 兼容平台：Claude Code, OpenClaw, Codex

## 反转类型分类

### 1. 设定反转

**定义**：把看似不可能的设定变成可能，或把可能的设定变成不可能

| 原设定 | 反转后 | 案例 |
|--------|--------|------|
| 密室锁门 | 密室没锁 | 《名侦探的守则》 |
| 门从内锁 | 从外锁 | 机械装置 |
| 窗户关闭 | 窗户打开 | 利用外部工具 |
| 不可能进入 | 有第三通道 | 建筑结构 |

### 2. 身份反转

**定义**：凶手的身份与读者认知相反

| 原设定 | 反转后 | 案例 |
|--------|--------|------|
| 双胞胎 | 三胞胎 | 《分裂的推理》 |
| 男性凶手 | 女性凶手 | 化妆/变性 |
| 熟人作案 | 陌生人 | 随机杀人 |
| 受害者 | 凶手 | 《嫌疑人X》 |
| 侦探是旁观者 | 侦探是参与者 | 叙诡 |

### 3. 手法反转

**定义**：犯罪手法与表面相反

| 原设定 | 反转后 | 案例 |
|--------|--------|------|
| 他杀 | 自杀伪装他杀 | 《嫌疑人X》 |
| 自杀 | 他杀伪装自杀 | 伪造成自杀 |
| 意外 | 谋杀 | 利用意外 |
| 谋杀 | 意外 | 故意制造意外 |

### 4. 动机反转

**定义**：动机与表面相反

| 原设定 | 反转后 | 案例 |
|--------|--------|------|
| 情杀 | 财杀 | 伪装成情杀 |
| 仇杀 | 意外 | 无差别杀人 |
| 有动机 | 无动机 | 随机杀人 |

### 5. 组合反转

**定义**：多重反转叠加

- 密室+不在场证明
- 身份+动机反转
- 叙诡+密室+时间反转（三重反转）

## 反转设计流程

### 步骤1：确定"表面真相"

先设计一个看起来"显而易见"的真相：

- 密室 → 门锁着
- 不在场证明 → 证人在现场
- 身份 → 凶手是A

### 步骤2：寻找反转点

对每个"表面真相"提问：

- 如果门没锁会怎样？
- 如果证人看错了会怎样？
- 如果A不是凶手会怎样？

### 步骤3：设计支撑条件

让反转成立：

- 门为什么没锁？（心理暗示/机械故障）
- 证人为什么看错？（注意力盲区/故意误导）
- A为什么不是？（有不在场证明/动机错误）

### 步骤4：检查公平性

- 反转的线索是否藏在文本中？
- 反转是否有足够铺垫？
- 是否符合动机合理性？

## 反转设计技巧

### 技巧1：预设答案

```
表面：A是凶手（有动机+机会）
反转：A不是凶手，真正的凶手是B
线索：A有不在场证明（藏在中段）
```

### 技巧2：多重烟雾

```
表面：有三个嫌疑人（A、B、C）
反转：凶手是第四个人D
线索：前三个都是烟雾弹，但各有"疑点"
```

### 技巧3：时间差反转

```
表面：死亡时间在凌晨
反转：死亡时间在昨晚，但尸体被移动
线索：尸斑位置与死亡时间不符
```

### 技巧4：空间差反转

```
表面：凶案发生在室内
反转：实际发生在室外，尸体被移入
线索：血迹形态不符
```

## 反转设计检查清单

- [ ] 表面真相是否足够"显而易见"？
- [ ] 反转是否有逻辑支撑？
- [ ] 线索是否隐藏在文本中？
- [ ] 反转是否超出读者预期但又合理？
- [ ] 动机是否合理？
- [ ] 是否有足够的铺垫？

## 高级反转技巧

### 1. 叙诡反转

利用叙述视角制造反转：

- 第一人称叙述者隐瞒身份
- 性别/年龄的暗示性描述
- 利用"读者以为"的常识

### 2. 结构反转

利用章节结构制造反转：

- 前半部分暗示A是凶手
- 后半部分用新证据证明B是凶手
- 结尾揭示AB都不是

### 3. 伏笔反转

设计"看似无用"的伏笔：

- 某个细节当时不重要
- 结尾时揭示是关键线索
- 需要读者重读才能发现
